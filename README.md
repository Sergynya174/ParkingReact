## React-приложение

Приложение поиска свободных мест у дома для парковки.

**Функционал:**

- Отображение порковочных мест на карте полученных с бэка.
- Геолокация.
- Поиск адресов.

**Технологии:**

- React
- Redux
- Axios
- Яндекс API карты
